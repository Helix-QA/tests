### Основные проверяемые области и документы (группировка по смыслу):
**Клиенты / Услуги**
- Акт оказания услуг → Акт, Счёт-фактура, Счёт на оплату, Статистика услуг
- Персональные занятия → Реестр посещаемости, Акт выполненных работ
- Членства / Пакеты услуг → Список доступных занятий по абонементу

**Продажи / Реализация**
- Обычная реализация товаров и услуг →  
  Акт выполненных работ  
  Расходная накладная  
  ТОРГ-12  
- Счёт на оплату (отдельный документ)

**Справки в налоговую** (по физкультурно-оздоровительным услугам)
- КНД 1151160 (основная, чаще всего ФОУ)
- (закомментированы 1151158 и 1151156 — возможно, были нужны раньше)

**Касса / Деньги**
- Поступление наличных → ПКО (приходный кассовый ордер)
- Возврат покупателю → КО-2 (расходный) + КМ-3 (акт о возврате сумм)

**Зарплата / Начисления**
- Начисление зарплаты сотруднику-тренеру  
  → Платёжная ведомость  
  → Расчётные листки  
  → Расшифровка начисления (обычная и подробная)

**Склад / Запасы** (очень большая часть теста)
- Заказ поставщику
- Поступление → М-4 (приходный ордер), Приходная накладная
- Возврат поставщику
- Инвентаризация → Инвентаризационная опись (ИНВ-3), сличительная ИНВ-19, ИНВ-3 пустая
- Оприходование → Акт об оприходовании
- Списание → Акт о списании (два варианта: обычный и с себестоимостью)
- Перемещение → Накладная на перемещение (обычная и по типу цен)
- Отчёт комитенту → Отчёт о продажах товаров комитента

**Отчёты**
- Расшифровка начислений и удержаний по конкретному сотруднику (из вариантов отчётов)

### Что важно помнить:
- Тест **очень сильно завязан на эталонные макеты** (сравнение «по шаблону»)
- Используются **фиксированные ссылки** на уже созданные в базе документы (ref=873c8a88... и подобные)
- Много данных создаётся в начале через вспомогательные сценарии (Я создаю ...)
- В конце происходит **очистка** организации, структурной единицы и помещения
- Тест **не про функциональность**, а именно про **визуальное соответствие печатных форм эталонам**

Коротко одним предложением:  
**Проверяем ~35 печатных форм по клиентам, продажам, кассе, зарплате и складу в типичной конфигурации фитнес-клуба/спортивной организации.**
