## Общее назначение сценария

Данный набор сценариев предназначен для **автоматической проверки загрузки и обновления различных классификаторов** в системе 1С с использованием Vanessa Automation.
Сценарии имитируют действия пользователя: переход по разделам, запуск загрузки данных, выбор элементов и проверку успешного завершения операций.

Перед выполнением тестов система приводится в «чистое» состояние — открывается тестовый клиент и закрываются все лишние окна.

---

## Сценарий «Банки»

**Что выполняется:**

1. Открывается справочник **«Классификатор банков»**.
2. Запускается мастер загрузки классификатора.
3. Устанавливаются флажки для загрузки данных.
4. Выполняется обновление классификатора.
5. Ожидается сообщение об успешном завершении загрузки (до 150 секунд).

**Проверка:**

* Текст результата строго соответствует сообщению
  *«Обновление классификаторов успешно завершено.»*
* Форма закрывается.

---

## Сценарий «Валюты»

**Что выполняется:**

1. Открывается справочник **«Валюты»**.
2. Выполняется подбор валют из классификатора.
3. Выбирается первая валюта в списке.
4. Дважды запускается загрузка курсов валют.
5. Выполняется переход между вкладками и реквизитами формы.

**Назначение:**

* Проверка корректности загрузки и обработки курсов валют.

---

## Сценарий «Диагнозы»

**Что выполняется:**

1. Открывается **классификатор диагнозов**.
2. В таблице выделяются все записи.
3. Запускается команда обновления.

**Проверка:**

* Ожидается, что активным станет окно
  **«Классификатор диагнозов»** в течение 30 секунд.

---

## Сценарий «Должности»

2. Выделяются все записи в списке.
3. Выполняется подбор данных из классификатора.

  **«Классификатор должностей»**.

## Сценарий «Единицы измерений»

**Что выполняется:**

**Проверка:**

  **«Общероссийский классификатор единиц измерения»**.

---

## Сценарий «НМУ»

*(Номенклатура медицинских услуг)*

**Что выполняется:**

1. Открывается классификатор медицинских услуг.
2. Запускается загрузка классификатора.
3. Последовательно раскрывается иерархия услуг:

   * группа услуг;
   * подгруппа;
   * конкретная услуга.
4. Для выбранной услуги:

   * активируется поле выбора;
   * устанавливается флажок;
   * изменения сохраняются.

**Проверка:**

* Выполняется обновление классификатора.
* Ожидается окно
  **«Классификатор номенклатуры медицинских услуг»**.

## Сценарий «Страны»

**Что выполняется:**

1. Открывается справочник **«Страны мира»**.
2. Запускается классификатор ОКСМ.
3. Выбирается первая страна.
4. При наличии ссылки выполняется дополнительное действие.
5. Окно классификатора закрывается.

**Проверка:**

* Выполняется повторный выбор страны из классификатора.
* Проверяется открытие окна
  **«Страны мира»**.

---

## Итог

Данный код:

* последовательно **проверяет загрузку, обновление и корректное открытие всех ключевых классификаторов**;
* контролирует успешность операций через сообщения, активные окна и сравнение с эталонными данными;
* обеспечивает уверенность в том, что классификаторы работают стабильно и корректно после обновлений.
